### Setup

**.env** requirements:

### Running the app

* npm start
* Endpoint: /food
    * Returns Object

> {
>   
>  "foodName": "ice cream",
>
>}

* Endpoint: /clothes
    * Returns Object

> {
>   
>  "clothingName": "dress",
>
>}

### Tests
* Unit Tests: npm run test
* Lint Tests: npm run lint
